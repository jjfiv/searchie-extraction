# searchie-extraction
Feature-Extraction code for "Named Entity Recognition with Extremely Limited Data" from the LND4IR workshop.

## Building the code

Requires Java 8 and Apache Maven (probably >3).

    mvn install

## Running the code

    mvn exec:java
